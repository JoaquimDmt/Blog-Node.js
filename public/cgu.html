Blog en Node.js par Joaquim Dimitrov

Voilà ma première expérience avec Node.js, un blog comprenant les quatres opérations de base de la persistance des données : create, read, update, delete. Directement lié à une base de données MongoDB, il est possible d'y consulter et rédiger des articles. Un espace administrateur est également présent pour pouvoir modifier ou supprimer n'importe quel article.

Les technologies utilisées pour ce blog ont donc été :

  • Node.js
  • MongoDB
  • NPM
  • Express.js
  • Pug.js

Vous pouvez retrouver mon travail librement à disposition sur Github : https://github.com/JoaquimDmt/Blog-Node.js
